# 자동차 경주 게임
## Step3. 자동차 경주
#### 기능 구현 목록
* 사용자로부터 자동차 대수 입력받는 기능
* 사용자로부터 시도 횟수 입력받는 기능
* 0-9 사이의 random 값 획득 기능
* random 값이 4 이상 여부 확인 기능
* 4 이상인 경우 자동차 이동시키는 기능
* 자동차 상태 화면 출력 기능

#### 요구사항
* 모든 로직에 단위 테스트를 구현 (UI기능 제외)
* 자바 코드 컨벤션
* else 예약어 쓰지 않기
* README.md 작성 
* commit메시지는 기능 목록 단위로 구분
